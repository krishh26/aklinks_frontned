Exchange Rate Configuration

Set the exchange rate for converting 1 US Dollar (USD) to Indian Rupees (INR). This rate will be used throughout the application for currency conversions.

Current exchange rate: 1 USD = {{ exchangeRate }} INR

ℹ️ Information

  • The default exchange rate is set to 80 INR per 1 USD.
  • This rate affects all currency conversions throughout the application.
  • Only administrators can modify this setting.
  • Changes will be reflected immediately after saving.
